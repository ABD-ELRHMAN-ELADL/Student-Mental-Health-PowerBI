# 🎓 Student Mental Health & Academic Performance Dashboard

## 📌 Overview

This Power BI , Excel , Tablau project analyzes the relationship between students' academic performance, mental health, lifestyle, and social factors.

The dashboard helps identify the key factors affecting student success and supports data-driven decision-making in educational institutions.

## 🛠 Tools

- Tablau

## 📊 Features

- Executive Dashboard
- Academic Performance Analysis
- Mental Health Analysis
- Lifestyle Analysis
- Dropout Risk Analysis
